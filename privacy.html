Política de Privacidade

Última atualização: Janeiro de 2025

1. Dados coletados

AutoPost coleta apenas os dados necessários para publicar conteúdo nas plataformas conectadas:

DadoFinalidadeArmazenamento
Tokens OAuthAutenticar com as plataformasLocal, criptografado
Mídia (imagens/vídeos)Upload para publicaçãoTemporário, local

2. Compartilhamento de dados

Não compartilhamos, vendemos ou transferimos seus dados pessoais a terceiros. Os únicos servidores externos acessados são as APIs oficiais das plataformas sociais (TikTok, Instagram, YouTube, etc.) para realizar as publicações.

3. Armazenamento

Todos os dados são armazenados localmente no dispositivo do usuário. Tokens de acesso são criptografados com Fernet (AES-128-CBC). Não existem servidores próprios de armazenamento.

4. Direitos do usuário

Você pode a qualquer momento:

  • Revogar o acesso OAuth diretamente nas plataformas conectadas
  • Deletar o banco de dados local para remover todos os tokens

5. Contato

Para dúvidas sobre privacidade, acesse as configurações da ferramenta.
